本项目就是为书写各种文档提供骨架和范本，做一些对人类有意义的事。
个人身单力薄，所谓众人拾柴火焰高，欢迎大家向我们 `Pull Requests`。

### 要求

* Markdown 格式，方便维护和查看迭代。
* 使用 HTML 注释描述需要填写的内容说明，如：`<!-- 编辑状态才可以看见这些注释。 -->`
* 来源注明出处，尊重别人的劳动果实。

### 贡献奖励

* 按 `Pull Requests` 成功一次积一分算。
* 暂时没有物质奖励，如有 壕 赞助可以筹划。
